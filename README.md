# Supermarket Sales Performance Dashboard 🛒

## Project Overview
This project analyzes a quarter's worth of supermarket sales data to uncover performance trends across different branches, product categories, and customer demographics. The goal of this analysis is to provide actionable insights into revenue generation and customer purchasing behavior to help optimize inventory and marketing strategies.

## Tools & Techniques Used
* **Microsoft Excel:** Used for the entirety of the project.
* **Data Processing:** Data cleaning, formatting, and structural organization.
* **Data Analysis:** Utilized Pivot Tables to aggregate large datasets into readable metrics.
* **Data Visualization:** Built interactive Pivot Charts (Bar charts, Pie charts) compiled into a unified, single-view dashboard.

## Key Insights & Findings
Based on the Q1 sales data, several key trends emerged:

* **Overall Revenue:** The total sales across all branches for the recorded period amounted to **$322,966.75**.
* **Top Performing Branch:** The **Giza** branch led overall sales with $110,568.71, slightly outperforming the Alex and Cairo branches, which were nearly tied at ~$106K each.
* **Demographic Purchasing Power:** Female customers drove the majority of the revenue, accounting for **$194,671.84** (roughly 60% of total sales), compared to male customers at $128,294.91.
* **Product Line Popularity:** **Food and beverages** was the highest-grossing category ($56,144.84), closely followed by Sports and travel. Health and beauty generated the lowest revenue for this period.
